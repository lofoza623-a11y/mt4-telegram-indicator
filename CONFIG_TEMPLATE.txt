//|                     BUFFER NUMBER REFERENCE                       |
//+------------------------------------------------------------------+

Most indicator buffers follow these patterns:

STANDARD ARROW INDICATORS:
Buffer 0 = Buy signal (up arrow)
Buffer 1 = Sell signal (down arrow)

OSCILLATORS (RSI, STOCHASTIC, etc.):
Buffer 0 = Main line
Buffer 1 = Signal line (if applicable)
Custom signal indicators may use different buffers - check code

MOVING AVERAGE CROSSOVERS:
Buffer 0 = Fast MA
Buffer 1 = Slow MA
Buffer 2 = Buy signal (if indicator generates signals)
Buffer 3 = Sell signal (if indicator generates signals)

CUSTOM INDICATORS:
Always verify using Data Window (Ctrl+D) or check the indicator code

TO FIND BUFFER NUMBERS:
1. Add indicator to chart
2. Press Ctrl+D to open Data Window
3. Move cursor over a signal
4. Check which buffer shows a value (not EMPTY_VALUE or 0)
5. Buffer numbers are shown as "Indicator Name [0]", "[1]", etc.

//+------------------------------------------------------------------+
//|                     TROUBLESHOOTING TIPS                          |
//+------------------------------------------------------------------+

‚ùå NO SIGNALS DETECTED:
1. Verify indicator is on the chart
2. Check buffer numbers using Data Window (Ctrl+D)
3. Enable EnableDebugMode = true
4. Check MT4 Terminal (Ctrl+T) ‚Üí Experts tab for logs
5. Verify timeframe data is available

‚ùå WRONG SIGNALS:
1. Buy and sell buffers might be swapped
2. Test with EnableDebugMode = true
3. Verify buffer numbers in Data Window
4. Check if indicator uses EMPTY_VALUE or 0 for no signal

‚ùå TOO MANY DUPLICATE ALERTS:
1. Increase DuplicatePreventionBars value
2. Use "Next Candle" timing (1) instead of "Same Candle" (0)
3. Change ConfirmMode to All or Majority

‚ùå TELEGRAM NOT WORKING:
1. Verify TelegramBotToken (no spaces)
2. Verify TelegramChatID (no spaces)
3. Send a message to your bot first
4. Check internet connection
5. Enable EnableDebugMode = true for detailed logs

‚ùå BUFFER NUMBER ERRORS:
1. MT4 supports buffers 0-7 only
2. Check indicator code for SetIndexBuffer() calls
3. Use Data Window to verify correct buffers
4. Some indicators may not have signal buffers
